## Features

- **AI-powered news analysis**
- **Actionable startup ideas from trends**
- **Waitlist for early access**
- **Dark theme editorial design**

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions, Storage)
- **AI:** Claude / Gemini via API
- **Hosting:** Lovable
- **Build Tool:** Vite

## Getting Started

```bash
# Clone the repo
git clone https://github.com/LazyUnicornFounder/Breaking-Muse.git
cd Breaking-Muse

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your Supabase credentials to .env

# Start the dev server
npm run dev
```
